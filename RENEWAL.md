## 2. 신규 기능 목록 (대화에서 도출)

### A. 서가 홈 (Phase 2)

| # | 기능 | 구현 포인트 |
|---|------|------------|
| A1 | **사용자 정의 그룹(선반)** | 하드코딩 `DOC_GROUPS`(참고문헌)를 일반 그룹으로 일반화 — 그룹명·소속 문헌·순서. 저작 모드 그룹 편집 UI("그룹 편집" 버튼). 선반 접기 상태는 localStorage |
| A2 | **문헌 표지 색** | `DocJSON.color?` 옵션 필드. 미지정 시 팔레트 8색(tokens.md) 순환 자동 배정. lint에 값 형식 검사 |
| A3 | **이어서 학습** | 마지막 학습 위치(docId·level·mode·진행 인덱스) localStorage 저장 → 홈 히어로에서 원터치 복귀. 학습 이력 없으면 히어로 숨김 |
| A4 | **문헌 상세 오버레이** (=계획했던 "문헌 목록창" 흡수) | 참고문헌 보유 문헌(무더기) 클릭 시: 미니 표지 + 모드 시작(안키/순차) + 레벨별 장수 칩 + 참고문헌 미니 표지(오버레이 안에서 숫자 키 1~N 재부여) + Esc/바깥 클릭 닫기. 참고문헌 없는 문헌은 기존처럼 바로 mode 화면으로 (오버레이 생략) |
| A5 | **streak 홈 노출** | 기존엔 안키 결과 화면에만 → 홈 헤더 필("연속 학습 N일") |
| A6 | **키보드 배지** | 표지 우상단 1~9 배지 — 기존 단축키 유지 시각화 |
