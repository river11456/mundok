## 2. 서체 — 3역할 체계

명조(Noto Serif) **전면 제거**. 위계는 서체 교체가 아니라 굵기·크기로만.

| 역할 | 스택 | 규칙 |
|------|------|------|
| UI · 한글 · **학습 본문 한자** (고딕) | `"Pretendard Variable",Pretendard,"Noto Sans KR","Noto Sans TC","Noto Sans SC",-apple-system,"Apple SD Gothic Neo",sans-serif` | 같은 뼈대(Source Han) 지역 폴백 사슬 — 벽자가 KR→TC→SC로 넘어가도 획 스타일 유지. 자형 우선순위: 한국→전통→간체 |
| 해서 (디스플레이) | `"LXGW WenKai TC","LXGW WenKai"` + 위 고딕 폴백 | **홈 워드마크 · 표지 제첨 · char 카드 대형 한자 전용.** 16px 미만 금지, 학습 문장 본문 금지 (대만 자형·커버리지 리스크) |
| 굵기 위계 | 본문 400~500 · 제목 600~700 · 통계 800 | 숫자는 `font-variant-numeric:tabular-nums` |

## 3. 형태

| 요소 | 라운드 |
|------|--------|
| 리스트 카드·히어로 | 18~20px |
| 학습 카드 | 24px |
| 통계 타일 | 18px |
| 테이블 카드 | 20px |
| 버튼·칩·필 | 999px (알약) |
| 표지 | `6px 12px 12px 6px` (책 등 쪽만 덜 둥글게) |

표지 규격: 비율 `3 / 4.1`, **플랫**(스파인·입체 음영 없음, 외부 그림자만), 제첨(題簽 — 세로쓰기 해서 라벨) 좌상단, 키보드 숫자 배지 우상단, 장수 좌하단, 최근 학습 칩 우하단.
참고문헌 = **겹침 무더기 + 오버레이**: 참고문헌 있는 문헌은 뒤에 여러 권이 자연스럽게 겹친 무더기(뒤 책 = 참고문헌 각각의 표지색, 우상향 오프셋 + 교대 회전 2.4/-1.6/0.8deg, 호버 시 부채꼴로 살짝 벌어짐) → 클릭 시 **문헌 상세 오버레이**(블러 백드롭 + pop 애니메이션) — 미니 표지·모드 버튼·레벨별 장수 칩·참고문헌 미니 표지(오버레이 내 숫자 배지 재부여). 이 오버레이가 "문헌 목록창" 기능을 겸한다. Esc/바깥 클릭 닫기.

## 4. 모션

- 호버 스프링: `transform .2s cubic-bezier(.34,1.56,.64,1)` — 카드 `-2px`, 표지 `-4px + rotate(-.6deg)`, 버튼 `-1px`
- 프로그레스 바: 높이 5px, 채움 트랜지션 `.3s`
- 색·테두리 전환: `.15~.2s`
